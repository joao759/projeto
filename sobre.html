Linguagem de programação

Nosso site utilizou por base a linguagem HTML, que é umas das principais linguagens de programação para sites e que tivemos por mais fácil manuseio, ainda mais utilizando de recursos BootsTrap. A parte visual foi utilizado o CSS, que é uma linguagem de estilos para editar o design do site com cores, sombras e outros. E por último temos o JavaScript, que foi de extrema importância para termos sucesso em implementar a nossa ferramentar de "adaptar visão". O JavaScript teve por função realizar as trocas de imagens e cores para o tipo de daltonismo selecionado.

Ferramentas

Para fazer este site utilizamos de algumas ferramentas e recursos que foram de extrema importância para nós e fizeram com que a criação da nossa ideia pudesse ser a mais organizada e funcional possível para todo mundo. No caso destacamos três principais ferramentas que mais utilizamos. Além disso também vamos destacar alguns outros sites e extensões que nos ajudaram a ter uma ideia base de como fariamos o projeto.

+

VS Code

Visual Studio Code é um editor de códigos que possui de diversas extensões e recursos que ajudam muito na programação. Ele possui controle Git incorporado (outra ferramenta muito importante para a construção desse projeto), complementação de código, refatoração de código e vários outros.

+

BootsTrap

É um framework utilizado para o desenvolvimento de componentes de interface e front-end (parte frontal do site; que interage com o usuário) que tem por função auxiliar na criação de sites que sejam mais amigáveis com o usuário e também responsivos (possa se adequar a quaquer tipo de tela. Parte muito importante de um site).

+

GitHub

O GitHub foi de muito ajuda para que não ocorresse de precisar gastar horas e horas tentando entender o que foi editado no código-fonte por um outro colega. O GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão que permite programadores trabalhar em conjunto através de recursos de fácil comunicação.

Referências

Para realizar nosso projeto utilizamos de alguns sites e extensões para obter uma boa base. A partir deles conseguimos produzir conteúdos consistentes sobre o daltonismo, tais como: como ocorre, qual é a sua incidência, quais são os principais tipos de daltonismo, o que eles afetam, como veem as cores e dentre outros.

Um site que foi de grande utilidade para nós foi o Coblis. Ele é um simulador de daltonismo em que pode ser inserido uma imagem para simula-la. Utilizamos esse site com grande regularidade para conseguirmos colocar em funcionamento a nossa ferramenta de adaptação de visão. Também há extensões do chrome que simulam o daltonismo e são muito interessantes para experimentar e entender a forma de como funciona a mudança de cores de acordo com o tipo de daltonismo. Alguns deles caso queira provar: Color Enhancer, Spectrum e por último Colorblind.
